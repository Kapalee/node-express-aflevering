# Node, express opgave

Download appen, åben den og åben terminalen

I terminalen kør:
npm install express
npm install cors

Derefter i terminalen sørg for du er i backend mappen "cd backend"

Når du er sikker på du står i backend, start dermed appen "npm start"

Det starter backend-serveren og du kan nu starte en live-server ved at højreklikke på index.html og klikke "open with live server"
